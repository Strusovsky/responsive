Responsive

and steps to apply it to your site

What it is?

Responsive is web-design approach that allows website looks good on any device, resolution and screen orientation.
In this case we say website responds to all conditions.

Step 2

Don't forget to add viewport meta tag to head

<meta name="viewport" content="width=device-width, initial-scale=1" />

Step 3

Define horizontal screen size breakpoints. Create mixins or just take them from ApplePie

      $break-small: 480px
      $break-medium: 768px
      $break-large: 980px
      $break-x-large: 1200px
      
      =screen($size)
        @if $size == big-phone-up
          @media only screen and (min-width: $break-small)
            @content
        @if $size == tablet-up
          @media only screen and (min-width: $break-medium)
            @content
        @else if $size == desktop-up
          @media only screen and (min-width: $break-large)
            @content
        @else if $size == big-desktop
          @media only screen and (min-width: $break-x-large)
            @content
        @else if $size == phone-only
          @media only screen and (max-width: $break-medium - 1)
            @content
        @else if $size == small-phone-only
          @media only screen and (max-width: $break-small - 1)
            @content
        @else if $size == big-phone-only
          @media only screen and (min-width: $break-small) and (max-width: $break-medium - 1)
            @content
        @else if $size == tablet-only
          @media only screen and (min-width: $break-medium) and (max-width: $break-large - 1)
            @content
        @else if $size == desktop-only
          @media only screen and (min-width: $break-large) and (max-width: $break-x-large - 1)
            @content
        @else if $size == print
          @media print
            @content

Step 4

Use at least one container to which breakpoints will be applied.

<div class="wrapper"> ... </div>

Step 5

Don't create separate CSS file with responsive styles. Use mixins right in the place where CSS-object is described.

        .wrapper
          width: 946px
          margin: 0 auto
          
          +screen(tablet-only)
            width: 746px
          
          +screen(phone-only)
            width: 446px

Step 6

It's better to clear columns properties for the smallest devices and resolutions.

        .col1of2
          width: 50%
          float: left
          +screen(small-phone-only)
            width: 100%
            float: none
